# Aoi.js 5.0.5 and node.js v16 bot!

A nice infrastructure that you can use to build your Aoi.js bot!

# Required packages 

```
npm i aoi.js
```
```
npm i discord.js
```
```
npm i express
```
```
npm i ytdl-core
```
```
npm i ytdl-core-discord
```
